# Code of Conduct

## Our standard

We want Bumpkin to be a project where people can ask basic questions, disagree honestly, and contribute without getting piled on.

Please be:

- respectful
- direct without being hostile
- patient with newer contributors
- open to feedback

## Unacceptable behavior

This includes:

- harassment
- personal attacks
- discriminatory language
- doxxing
- bad-faith intimidation
- repeated disrespect after being asked to stop

## Scope

This applies in:

- issues
- pull requests
- discussions
- project chat spaces tied to Bumpkin

## Reporting

If someone is making the project unsafe, please report it privately through GitHub security/contact channels used by this repo.

## Enforcement

Project maintainers may remove comments, close threads, or block participation when behavior crosses the line.

The goal is not punishment for small mistakes.
The goal is to keep the project workable and safe for contributors.
